Guide des Commandes

Voici comment écrire vos commandes :

  • left : Tourner une fois à gauche.
  • right : Tourner une fois à droite.
  • up : Avancer d'une case vers le haut.
  • down : Avancer d'une case vers le bas.
  • 3*left : Tourner trois fois à gauche.
  • 2*up : Monter de deux cases.

Utilisez "*" pour répéter une commande plusieurs fois.
